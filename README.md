# Language Definition

```c++
// OPERATORS
+   // addition
-   // subtraction
*   // multiplication
/   // division
%   // modulus

// SINGLE IMPORTANT CHARACTERS
:   // function call

// VARIABLE DECLERATIONS
c = 'c'     // evals to char
f = 0.0     // evals to float
i = 0       // evals to int

// DATA TYPES
char
float
int

// EXPRESSION RULES
var = 'c' + 0     // ASCII decimal number 'c' plus 0
var2 = 0.0 + 0     // Force integer to floating point

// FUNCTION DECLERATION
func someFunc: a, b, c 
{
    ret a + b + c
}

// OR

func someFunc2
{
    
}

// CALLING FUNCTIONS
:someFunc2

:give_me_some_pie

```
